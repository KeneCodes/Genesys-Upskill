My Roadmap & Personal Development 📈

Web Development Journey

My goal is to transition from foundational understanding to full-stack proficiency through a structured four-phase journey. I begin with Phase 1: The foundations, focusing on HTML5 and CSS3 to build solid semantic structures. In Phase 2, I am mastering Version Control & Deployment using Git and GitHub to manage my workflow. Moving into Phase 3, I will tackle Interactivity through JavaScript and DOM manipulation, finally in Phase 4: I will focus on Advanced Architecture, where I will integrate responsive design and APIs to create complex, user-centric applications.

My Daily Developer Routine

  1. 07:00 AM: My Golden Hour. This is when I feel like a genius who can solve any problem. It usually lasts until about 7:15 AM though 😁.
  2. 10:00 AM: The Research/Work phase. I call it the "Why is my code doing exactly what I told it to do, but not what I wanted it to do" phase ✨.
  3. 02:00 PM: Project (Personal). I finally start typing or learning or any other project I have set out. This is much more satisfying than staring at a blank screen. I’m mostly trying to make things look like they belong on the internet.
  4. 08:00 PM: Unwind/ Me-time. I put the laptop away before it starts staring back at me. I pick up the yarn and crochet. It’s the only time of day where if I make a mistake, I don't get an "unexpected token" error... just have to frog the piece 😩.
